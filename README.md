# 📊 Call Center Data Analytics Dashboard 

## 📌Overview
This project is an interactive **Call Center Data Analytics Dashboard** built using **Power BI**.  
It helps analyze call center operations by tracking key performance metrics such as total calls, call duration, and resolution rates.

The dashboard provides insights into "agent performance, team efficiency, and regional trends", enabling better decision-making.  
It helps identify patterns, improve customer service quality, and optimize overall call center performance.

This project also demonstrates skills in data cleaning, data modeling, SQL querying, and DAX calculations.

---

## 📌Key Metrics
- Total Calls
- Average Call Duration
- Call Resolution Rate
- Agent Performance
- Region-wise Analysis
